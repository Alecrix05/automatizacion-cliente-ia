# Automatización de Atención al Cliente con IA

Proyecto desarrollado utilizando:

- n8n
- Docker
- WSL2
- OpenRouter
- Modelos LLM
- Webhooks

## Flujo

Cliente → Webhook → IA → Respuesta

## Objetivo

Automatizar respuestas a clientes para servicios de soporte técnico de computadoras.

## Tecnologías

- Ubuntu (WSL2)
- Docker
- n8n
- OpenRouter
- JSON
- REST APIs
